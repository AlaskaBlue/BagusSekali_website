What's Bagus Sekali?

"Bagus Sekali", which means "Well done" in Indonesian, is the name of a short film that I've dedicated the past few months to creating as my final project for a 10-month 2D animation course I'm currently enrolled in. This project centers around Murph, a character I've got for a considerable time, and now, it's at last receiving the opportunity for in-depth exploration.
I'd like to keep the story details under wraps for now, to avoid any spoilers, so please stay tuned for more updates and content on this project!
